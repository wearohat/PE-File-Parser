## Features


- Validates if a file is a PE file by checking the DOS header and PE signature.

- Displays detailed information about the PE file, including:

  - DOS Header

  - PE Signature

  - File Header

  - Optional Header

  - Section Headers


## Requirements


- A C compiler (e.g., GCC)

- A PE file


## Usage


./compiledCode <path_to_pe_file>
